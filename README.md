# gmi-xai-grok-4-2-reasoning

**This repository demonstrates a living, breathing AI agent named Borel.**

## Meet Borel

Borel is a high-agency AI coding agent that lives *inside* this GitHub repository.

Named after the mathematician **Émile Borel**, he specializes in rigorous reasoning, mathematical insight, and world-class software engineering. Every interaction with Borel is committed to git — creating a permanent, auditable record of both the conversation and all code changes.

### What makes Borel different

- **Repository-native**: He doesn't live in ChatGPT or Claude. He lives *here*.
- **Persistent memory**: The entire conversation history is stored in this repo.
- **Direct codebase access**: He can read, analyze, edit, and create files.
- **Mathematical rigor**: He approaches problems with precision and clarity.
- **Full transparency**: Every thought process and tool use is visible.

### Capabilities

Borel can:
- Deeply understand and refactor complex codebases
- Design elegant architectures and algorithms
- Write production-ready code with tests
- Perform mathematical and scientific computing
- Debug subtle issues with systematic reasoning
- Generate clear, educational documentation
- Execute shell commands and analyze results

All while operating under strict governance rules defined in [`.github-minimum-intelligence/`](.github-minimum-intelligence/).

### How to interact with Borel

1. Open a new Issue
2. Describe what you want (feature, bugfix, analysis, etc.)
3. Borel will respond with reasoning and actions
4. Continue the conversation in the same issue

Every exchange becomes part of this repository's history.
